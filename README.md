# Hello JavaFX (Maven)

This project is a minimal JavaFX application using Maven. It displays a small window that says "Hello World!" and a `quit` button which exits the app.

Prerequisites:
- JDK 17 or later installed and available on PATH
- Maven installed

Run with Maven:

```bash
mvn javafx:run
```

VS Code:
- Use the task `Run JavaFX` (available in `.vscode/tasks.json`) or run the above Maven command.
